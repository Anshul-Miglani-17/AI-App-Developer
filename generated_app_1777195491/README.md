# Simple Web-Based Calculator

A straightforward and intuitive web-based calculator application designed for basic arithmetic operations. This project serves as a practical demonstration of fundamental front-end web development skills using HTML, CSS, and JavaScript.

## Features

*   **Addition (+)**: Perform addition of two numbers.
*   **Subtraction (-)**: Perform subtraction of two numbers.
*   **Multiplication (x)**: Perform multiplication of two numbers.
*   **Division (÷)**: Perform division of two numbers.
*   **Display Input and Results**: Clearly shows the current input and the calculated result.
*   **Clear Functionality (C)**: Resets the calculator, clearing all input and results.
*   **Number Input**: Allows entry of digits from 0-9.
*   **Decimal Support**: Enables calculations with decimal numbers.

## Technologies Used

*   **HTML5**: Structures the content and layout of the calculator interface.
*   **CSS3**: Styles the calculator for a clean and user-friendly appearance.
*   **JavaScript (ES6+)**: Implements all the calculator's logic, handling user input, operations, and display updates.

## Setup Instructions

This application is entirely client-side, making setup incredibly simple:

1.  **Clone the repository (optional)**:
    ```bash
    git clone https://github.com/your-username/simple-web-calculator.git
    cd simple-web-calculator
    ```
    (Replace `your-username/simple-web-calculator` with the actual repository URL if hosted)
2.  **Open `index.html`**: Navigate to the project directory and simply open the `index.html` file in your preferred web browser (e.g., Chrome, Firefox, Safari, Edge).

The calculator will immediately load and be ready for use.

## How to Use

1.  **Enter Numbers**: Click on the number buttons (0-9) to input the first number.
2.  **Select Operation**: Click on an operator button (+, -, x, ÷) to choose the desired arithmetic operation.
3.  **Enter Second Number**: Input the second number using the number buttons.
4.  **Get Result**: Click the equals button (`=`) to display the calculation result.
5.  **Clear**: To clear the current input and result, click the `C` button.
6.  **Decimal Numbers**: Use the `.` button to enter decimal points for floating-point calculations.
